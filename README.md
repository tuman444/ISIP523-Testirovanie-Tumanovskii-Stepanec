Использовали приложение *Visual Studio 2022*. Шаблон приложения: *Консольное приложение Wpf(.NET Framework) на языке программирования С#.*

---
**Архитектура приложения состоит из 4 страниц:**

  - 1 страница - выбор примера;
  - 2-4 страницы - сами примеры.

**Также к каждой странице прописаны определенные функции, в зависимости от требований страницы:**

  - для первой - навигация между примерами;
  - для второй, третьей и четвертой - функций для подсчёта примера с данными перемеными, которые пользователь вводит с клавиатуры в ТехtBox.

> При нажатии на кнопку **"Очистить"** на странциах 2-4, данные введенные пользователем очищаются.
>
> При нажатии на кнопку **"Вычислить"** на странциах 2-4, приложение решает пример с определенными требованиями и вписывает ответ в соответствующую ячейку.
>
> При нажатии на кнопку **"Назад"** на странциах 2-4, страница приложения меняется на предыдущую (на первую).
>
> При нажатии на кнопку **"Выйти"** на странциах 2-4, приложение завершает свою работу.
